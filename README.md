# ✈️ Emirates AI Airline Passenger Satisfaction System

## 📌 Project Overview
This project is a Machine Learning + Deep Learning based system that predicts airline passenger satisfaction using real-world airline data. It also includes a Streamlit web application for interactive predictions.

---

## 🎯 Objective
To analyze and predict whether a passenger is satisfied or not based on factors like:
- Flight distance
- Departure delay
- Seat comfort
- Passenger profile

---

## 🧠 Problem Statement
Airlines need to understand passenger experience in order to improve service quality and customer satisfaction. This project helps in predicting satisfaction using AI.

---

## 🚀 Features
- Passenger Satisfaction Prediction
- Interactive Web Dashboard (Streamlit)
- Real-time input system
- AI-based decision making

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Machine Learning

---

## 📊 Machine Learning Model
A classification model (e.g., Random Forest) is trained to predict:
- 1 → Satisfied Passenger 😊  
- 0 → Not Satisfied Passenger 😞  

The trained model is saved as `model.pkl`.

---

## 🌐 Web App
Built using Streamlit:
- User inputs passenger details
- Model predicts satisfaction instantly
- Simple and interactive UI
